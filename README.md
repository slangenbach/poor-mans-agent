# Poor man's agent

![CI](https://github.com/slangenbach/poor-mans-agent/actions/workflows/ci.yml/badge.svg)

An almost barebones agent implemntation for the poor dev

## Prerequisites

- [uv][1]
- [Task][2]

## Installation

1. Make sure you meet the prerequisites
1. Setup the development environment: `task setup`

## Configuration

Edit the **.env** file in project root directory and set the following variables:

| Variable | Description | Note |
| --- | --- | --- |
| LOG_LEVEL | Set the logging level | Defaults to 'INFO'
